
## Magazines Front end

## Libraries
- NextJs
- MUI
- Axios, use query for API integration
- Zustand for state managment
- react-hot-toast for toast messages

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

## Modules
- User login
- User sugnup
- Magazines(list and subscription)
- Subscriptions (list and cancel)
